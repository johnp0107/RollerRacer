## ğŸš€ Getting Started

### Prerequisites

- [Unreal Engine 5](https://www.unrealengine.com/en-US/download)
- Visual Studio 2022 (with C++ and UE components installed)
- Git

### Setup Instructions

```bash
git clone https://github.com/johnp0107/RollerRacer.git
```

1. Open `RollerRacer.uproject` in Unreal Engine 5.
2. Let the engine compile and generate project files if needed.
3. Click **Build** in Visual Studio.
4. Run in editor (or package the game to test multiplayer).
5. To test multiplayer, launch multiple instances or run with `-server` and `-client` flags.
