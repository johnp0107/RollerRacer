# 🏎️ RollerRacer – 3D Racing Game in Unreal Engine 5

A 3D racing game prototype built using Unreal Engine 5 and C++. Race through checkpoints, track lap times, and experience smooth multiplayer gameplay with UE5’s networking system.

---

## 🎮 Features

- 🚗 Vehicle physics and responsive player controls
- 🕹️ Checkpoint and lap tracking system
- 🌐 Multiplayer support (host and join sessions)
- 🧱 Modular C++ structure (GameMode, PlayerController, GameInstance)
- 💡 Blueprint integration for rapid UI and game logic prototyping
- 🛠️ GitHub-based version control and clean project organization

## 🚀 Getting Started

### Prerequisites

- [Unreal Engine 5](https://www.unrealengine.com/en-US/download)
- Visual Studio 2022 (with C++ and UE components installed)
- Git

### Setup Instructions

```bash
git clone https://github.com/johnp0107/RollerRacer.git
```

1. Open `RollerRacer.uproject` in Unreal Engine 5.
2. Let the engine compile and generate project files if needed.
3. Click **Build** in Visual Studio.
4. Run in editor (or package the game to test multiplayer).
5. To test multiplayer, launch multiple instances or run with `-server` and `-client` flags.
